## 페이지 전환 흐름

```
login → (로그인 성공) → dashboard
dashboard → (로그아웃 클릭) → login
login → (회원가입 클릭) → register
register → (로그인 클릭) → login
```

## 대시보드 구성 요소

### 1. 상단 네비바
- 로고 (AdminPanel)
- 네비 링크: 대시보드 / 분석 / 사용자 / 설정 (현재 클릭 시 이동 없음, 샘플)
- 유저 아바타 + 이름 (이메일 @ 앞부분 표시)
- 로그아웃 버튼

### 2. 통계 카드 (4개)
| 항목 | 샘플 값 | 강조 색상 |
|------|--------|---------|
| 총 회원수 | 12,847 | Indigo |
| 오늘 방문자 | 1,293 | Emerald |
| 이번달 매출 | ₩4.2M | Amber |
| 활성 세션 | 847 | Pink |

### 3. 월별 방문자 차트
- 바 차트 형태 (CSS만으로 구현, 라이브러리 없음)
- 1~12월 샘플 데이터

### 4. 최근 활동 목록
- 유저 아바타 + 이름 + 액션 + 경과 시간
- 5개 샘플 항목

### 5. 퀵 메뉴 (4개)
- 회원 관리 / 리포트 / 알림 설정 / 보안 로그

# PD (판돌) 컴포넌트 구현

## 작업 일자
2026-02-25

## 개요
`src/Components/Contents/PD.jsx` 기존 플레이스홀더를 인터랙티브 태스크 보드로 완성.
할 일 추가/완료/삭제와 주간 목표 진행률을 포함한 생산성 대시보드 형태.

| 파일 | 설명 |
|------|------|

---

## PD 컴포넌트 구성 요소

### 1. 요약 카드 (3개)
| 항목 | 내용 |
|------|------|
| 남은 할 일 | 미완료 태스크 수 (동적) |
| 완료 | 완료된 태스크 수 (동적) |
| 진행률 | 완료 비율 % (동적) |

### 2. 전체 진행 바
- 완료율에 따라 너비가 변하는 그라디언트 바
- `transition: width 0.5s ease` 애니메이션

### 3. 할 일 추가 폼
- 텍스트 입력 + 우선순위 셀렉트 (높음 / 중간 / 낮음)
- 제출 시 목록 맨 아래에 추가

### 4. 할 일 목록
- ⬜ / ✅ 버튼으로 완료 토글
- 완료 시 strikethrough + opacity 감소
- 우선순위 뱃지 (색상 코딩)
  - 🔴 높음: 빨간 계열
  - 🟡 중간: 노란 계열
  - 🟢 낮음: 초록 계열
- ✕ 버튼으로 항목 삭제

### 5. 이번 주 목표 섹션
- 주간 커밋 목표 / 이슈 해결률 / 코드 리뷰 완료
- 각 목표별 개수 + 진행 바 표시
