# 대시보드 기능 작업내역

## 작업 일자
2026-02-25

## 개요
로그인 후 접근 가능한 샘플 어드민 대시보드 페이지 구현.
기존 글래스모피즘 다크 테마를 그대로 유지하면서 확장.

## 페이지 전환 흐름

```
login → (로그인 성공) → dashboard
dashboard → (로그아웃 클릭) → login
login → (회원가입 클릭) → register
register → (로그인 클릭) → login
```

## 대시보드 구성 요소

### 1. 상단 네비바
- 로고 (AdminPanel)
- 네비 링크: 대시보드 / 분석 / 사용자 / 설정 (현재 클릭 시 이동 없음, 샘플)
- 유저 아바타 + 이름 (이메일 @ 앞부분 표시)
- 로그아웃 버튼

### 2. 통계 카드 (4개)
| 항목 | 샘플 값 | 강조 색상 |
|------|--------|---------|
| 총 회원수 | 12,847 | Indigo |
| 오늘 방문자 | 1,293 | Emerald |
| 이번달 매출 | ₩4.2M | Amber |
| 활성 세션 | 847 | Pink |

### 3. 월별 방문자 차트
- 바 차트 형태 (CSS만으로 구현, 라이브러리 없음)
- 1~12월 샘플 데이터

### 4. 최근 활동 목록
- 유저 아바타 + 이름 + 액션 + 경과 시간
- 5개 샘플 항목

### 5. 퀵 메뉴 (4개)
- 회원 관리 / 리포트 / 알림 설정 / 보안 로그

## 인증 로직 (Mock)

실제 백엔드 없이 프론트엔드에서만 동작하는 샘플 인증:
- 이메일: 아무 유효한 이메일 형식
- 비밀번호: 6자 이상이면 로그인 허용
- 로그인 성공 시 `{ email, name }` 객체를 App.jsx의 `user` state에 저장
